# ServicePilot AI

Ein modularer KI-Mitarbeiter für Handwerksbetriebe.

## Features

- KI-Kundenchat
- Lead-Erfassung
- Automatische Zusammenfassungen
- Wissensdatenbank
- Unternehmensspezifische Konfiguration
- Dashboard

## Ziel

Ein System, das innerhalb weniger Minuten für jeden Handwerksbetrieb eingerichtet werden kann.
